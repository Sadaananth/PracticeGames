## Simple PingPong Game

This is a simple PingPong game using SFML. In this game, I used only the shapes and text objects. 

![bandicam 2023-03-23 21-42-17-822](https://user-images.githubusercontent.com/17174550/227223935-8be03591-c3da-4804-a45a-c5975c1afe85.gif)

## Features available:
* Mouse click control
* Key board navigator
* Reset the game
* Ball hit counter
